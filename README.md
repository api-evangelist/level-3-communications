# Level 3 Communications (level-3-communications)
Level 3 Communications was a multinational telecommunications and Internet service provider that operated one of the largest fiber networks in the world, delivering Tier-1 IP transit, managed services, and content delivery. In 2017 it was acquired by CenturyLink, which later rebranded the combined company as Lumen Technologies.
